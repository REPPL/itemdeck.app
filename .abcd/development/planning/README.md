High-level design goals, architecture, and requirements.

- [architecture/](./architecture/) - System design and technical architecture

## Purpose

This directory contains forward-looking design documentation:
- **What** we're building (vision, goals)
- **Why** we're building it (requirements, constraints)
- **How** it should work (architecture, patterns)

## What Belongs Here

- Vision documents
- Architecture overviews
- Requirements specifications
- Design constraints
- Version planning goals

## What Doesn't Belong Here

- Feature specifications → [roadmap/features/](../roadmap/features/)
- Implementation details → [implementation/](../implementation/)
- Decision records → [decisions/adrs/](../decisions/adrs/)
