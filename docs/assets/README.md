# Assets

Project assets including images, logos, and media files.

## Structure

- `img/` - Images and logos

## Usage

Reference assets relative to project root:

```markdown
![Logo](./docs/assets/img/logo.png)
```

## Guidelines

- Use SVG for logos and icons (scalable)
- Optimise PNG/JPG before committing
- Document asset sources and licences if applicable
