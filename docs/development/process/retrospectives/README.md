## Structure

Each milestone retrospective is a directory:

```
retrospectives/
└── v0.X.0/
    └── README.md       # Post-milestone reflection
```

## Retrospective Requirements

Each retrospective MUST include:
1. What Went Well
2. What Could Improve
3. Lessons Learned
4. Decisions Made
5. Metrics (features, tests, coverage)
6. Related Documentation
