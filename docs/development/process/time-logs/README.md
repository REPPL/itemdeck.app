# Time Logs

Time tracking for development work.

## Format

Create daily log files: `YYYY-MM-DD.md`

```markdown
# Time Log: YYYY-MM-DD

| Start | End | Duration | Task | Notes |
|-------|-----|----------|------|-------|
| 09:00 | 12:30 | 3.5h | Feature X | Completed core logic |
| 14:00 | 17:00 | 3h | Testing | Added 15 tests |

**Total: 6.5 hours**
**Milestone: v0.X.0**
```
