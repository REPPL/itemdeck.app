## Release Notes

### Added
- Card flip animation with 3D CSS transforms and Framer Motion
- Configuration system with Zod v4 schema validation
- Image fallback system with SVG placeholders (initials + generated colours)
- Keyboard navigation with roving tabindex pattern
- Reduced motion support respecting `prefers-reduced-motion`
- 90 unit tests covering all new functionality

### Changed
- Card component restructured as compound component (`Card`, `Card.Back`, `Card.Front`, `Card.Inner`)
- CardGrid now manages flip state with `maxVisibleCards` enforcement

### Technical Details
- `ConfigProvider` with localStorage persistence
- `useGridNavigation` hook for keyboard navigation
- `useReducedMotion` hook for motion preference detection
- `MotionProvider` wrapping Framer Motion's `MotionConfig`
- `ImageWithFallback` component with loading skeleton
