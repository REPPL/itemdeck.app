## Key Implementation Notes

1. **Overlay Store Pattern**: Edits stored separately from source data in Zustand
2. **Schema-Driven Forms**: Edit forms generated from v2 entity type definitions
3. **LocalStorage Persistence**: Edits persist across sessions
4. **Export/Import**: Users can backup and share their edits
