## Key Implementation Notes

1. **CSS-Only Tooltips**: No JavaScript library required; uses `:hover` and `:focus-within`
2. **Accessible**: Includes `role="tooltip"`, `aria-label`, and `tabIndex={0}` for keyboard navigation
3. **Responsive**: Positions to the right on desktop, below on mobile (< 640px)
4. **Theme-Aware**: Uses CSS custom properties for colours
