## Key Implementation Notes

### Memory Game Status Bar (F-068)
- Hide overlay during active gameplay (pure focus mode)
- Show stats in bottom-left footer on completion
- Position matches SearchBar styling

### YouTube Support (F-069)
- Auto-detect YouTube URLs in images array
- Thumbnail preview with click-to-play
- No schema changes required

### Image Validation (F-070)
- Filter unretrievable images from gallery
- Cache validation results
- Loading state during validation

### MyPlausibleMe Format (F-071)
- Strict URL: `github.com/{user}/MyPlausibleMe/data/{folder}`
- Simplified username + folder input
- Migration for existing sources

### Settings Redesign (F-072)
- Reduce from 5 tabs to 4: Quick, System, Appearance, Data
- Add settings search with fuzzy matching
- Move ViewModeToggle to floating toolbar

### User Documentation (F-073)
- Diataxis framework: tutorials, guides, explanation, reference
- Priority: getting-started, keyboard-shortcuts, search-filters
