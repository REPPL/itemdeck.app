## Key Implementation Notes

### Top Badge Field Selector

Replaced the boolean "Show Rank Badge" toggle with a flexible field selector:
- Users can select any suitable field for the top badge (order, rank, verdict, rating)
- "None" option hides the badge entirely
- `showRankBadge` is now derived from `topBadgeField !== "none"`
- Renamed "Unranked Text" to "Text if Empty" for clarity

### Collection Toast Redesign

Changed from small top toast to prominent centred modal:
- Added backdrop with blur effect
- Increased icon and text size for visibility
- Added "Click to dismiss" hint
- Better visual feedback for collection switching

### Documentation Synchronisation

Major audit and fix of documentation indices:
- Fixed broken research links in ADRs README (path corrections)
- Added missing ADRs (013, 019) to index
- Added missing research documents (R-009, R-010) to index
- Updated feature checkboxes to reflect actual completion status
