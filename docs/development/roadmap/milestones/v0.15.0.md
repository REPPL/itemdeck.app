## Key Implementation Notes

### F-107: Empty Collection Handling

When no collections exist:

1. **Settings Panel**
   - Disable "Sources" tab (greyed out)
   - Auto-focus "Add Source" tab
   - Show helpful guidance message

2. **Main Window**
   - Replace error message with friendly "No collections" state
   - Provide clear call-to-action to add a collection

3. **Trigger Conditions**
   - App startup with no collections
   - Deleting the last collection
   - Invalid collection state detected
