## Key Implementation Notes

### F-107: Empty Collection Handling

When no collections exist:
1. **Settings Panel**: Disable "Sources" tab, auto-focus "Add Source"
2. **Main Window**: Friendly "No collections" state with CTA
3. **Triggers**: App startup, deleting last collection

### F-109: Launch Screen with Logo

- Use logo from `docs/assets/img/logo.png`
- Update LoadingScreen component
- Add fade-in animation

### F-110: Keyboard Shortcuts Review

- Separate navigation (single keys) from actions (Ctrl+X)
- Add vim-style navigation (j/k/h/l)
- Centralise config in `src/config/keyboardShortcuts.ts`

### F-114: Update Checking

- Use GitHub API to check commit timestamps
- Subtle badge indicator (not intrusive)
- Background polling every 15 minutes

### F-115: Caching Transparency

- Simple indicators: Fresh (green), Stale (yellow), None (grey)
- Tooltip with human-readable cache age
- No complex controls (user preference: simple indicators)
