## Release Notes

### Added
- TanStack Query integration for data fetching with caching
- GitHub raw URL data source support (fetches from MyPlausibleMe repository)
- Standardised card data schema with Zod validation
- Service worker caching for offline viewing (PWA support)
- Offline indicator component
- Loading skeleton components
- Query error boundary for graceful error handling

### Changed
- Mock data replaced with TanStack Query data fetching layer
- Card components now use DisplayCard type with guaranteed imageUrl

### Technical Details
- 211 tests passing
- TypeScript strict mode compliance
- PWA manifest with Workbox runtime caching strategies
