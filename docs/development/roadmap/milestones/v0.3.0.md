## Draft Release Notes

### Added
- Theme system with light/dark/auto modes
- Layout presets (grid, list, carousel)
- State persistence to localStorage/IndexedDB
- Settings panel for user preferences
- Manual refresh button for data updates
- ARIA live regions for screen reader announcements
- Test coverage reporting with thresholds
- Shuffle cards by default with toggle
- Card info button for extended content modal
- Enhanced card front with glassmorphism overlay
- Dynamic card elevation system
- Card badges for status and categories
- Responsive typography scaling

### Fixed
- ESLint/TypeScript errors (deprecated Zod syntax, type safety)

### Changed
- SettingsContext replaced with Zustand store
