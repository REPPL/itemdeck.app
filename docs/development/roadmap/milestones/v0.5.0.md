## Overview

This milestone focuses on:

1. **Card Front Redesign** - Rank badge with gold/silver/bronze for top 3, device badge, prominent info button
2. **Touch Support** - Fix iPad drag & drop using TouchSensor
3. **Expanded Card View** - Animate from card position to centre with blur background
4. **Image Gallery** - Support multiple images per card with swipe navigation on iPad
5. **Navigation** - Admin button (Ctrl+A toggle), search button/bar, explorer sidebar with filters
6. **Settings** - Reorganise into tabs (System, Theme, Behaviour, Card)
7. **Visual Themes** - Full visual difference (fonts, borders, shadows) not just colours

## Key Implementation Notes

### iPad Touch Support

```typescript
// Add TouchSensor alongside PointerSensor
import { TouchSensor } from "@dnd-kit/core";

const sensors = useSensors(
  useSensor(PointerSensor, { activationConstraint: { distance: 8 } }),
  useSensor(TouchSensor, { activationConstraint: { delay: 250, tolerance: 5 } }),
  useSensor(KeyboardSensor, { coordinateGetter: sortableKeyboardCoordinates })
);
```
