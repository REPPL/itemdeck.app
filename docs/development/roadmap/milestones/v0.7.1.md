## Overview

This patch release addresses issues identified after v0.7.0:

1. **Settings Store Integration** - Three theme customisation settings were stored but never read by components
2. **Visual Refinements** - Logo sizing, image positioning, and badge width adjustments
3. **UX Improvements** - Overlay state management and icon-based controls

## Dependencies

## Release Notes

### Fixed

- moreButtonLabel setting now correctly applied to Verdict button
- autoExpandMore setting now auto-expands overlay on card open
- zoomImage setting now controls first image zoom behaviour

### Changed

- Logo sizing uses max-bounds instead of fixed width percentage
- First image zooms 3% and aligns to top centre
- Default moreButtonLabel changed from "Details" to "Verdict"
- Verdict button uses InfoIcon instead of ChevronIcon
- Acknowledgement button is now icon-only
- Rank badge placeholder width increased to 240px

### Improved

- All overlays reset to closed state when detail view closes
