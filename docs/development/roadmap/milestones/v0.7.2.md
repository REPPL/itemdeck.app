## Overview

This patch release addresses visual inconsistencies and adds polish:

1. **Close Button Unification** - All overlay close buttons now use consistent 48px styling
2. **Animation Options** - New flip animation style for Verdict overlay (default for retro theme)
3. **Rating Display Fix** - Correct score normalisation between 5-point and 10-point scales
4. **Image Robustness** - Graceful fallback when platform logos fail to load

## Technical Decisions

### Rating Normalisation
Changed default `max` from 10 to 5 and added normalisation formula `(score / max) * 10`. Most existing data uses 5-point scales.

### Whole Stars Only
Removed half-star character (⯨) due to inconsistent font rendering. Using `Math.round()` for clean display.

### State-Based Image Fallback
React `useState` + `onError` pattern provides clean integration and consistent cross-browser behaviour.
