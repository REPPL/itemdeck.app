## Overview

This milestone adds user-requested features and closes gaps from v0.8.0:

1. **Random Selection** - Limit displayed cards to X of Y with toggle and slider
2. **Accessibility** - Skip-to-content link and ARIA live regions
3. **Storage Enhancements** - Export/Import collection and re-cache images

## Implementation Summary

### Phase 1: Random Card Selection

- Added `randomSelectionEnabled` and `randomSelectionCount` to settings store
- Settings store migration v13 → v14
- Fisher-Yates shuffle applied before existing shuffle/sort logic in CardGrid
- UI in Settings > Cards > Layout tab with toggle and slider
- Shows "X of Y cards" when enabled

### Phase 2: Accessibility Improvements

- Added skip-to-content link (visually hidden, visible on Tab focus)
- Added ARIA live region to LoadingScreen (`role="status"`, `aria-live="polite"`)
- Screen readers announce loading progress

### Phase 3: Storage Enhancements

- Created `collectionExport.ts` with export/import utilities
- Export Collection button downloads collection as JSON
- Import Collection button (placeholder for future implementation)
- Re-cache Images button forces re-download of all images
- Wired collection caching into useCollection hook
