## Key Capabilities

### Collection Comparison
- Load two collections simultaneously
- Find duplicates (exact ID match)
- Find similar items (fuzzy title matching)
- Identify unique items in each collection
- Export comparison results

### Matching Algorithms

| Type | Confidence | Method |
|------|------------|--------|
| Exact | Highest | Same entity ID |
| Title | High | Normalised title comparison |
| Fuzzy | Medium | Fuse.js similarity scoring |
| Field | Medium | Matching key field values |

## Key Implementation Notes

1. **Multi-Collection State**: Manage two collections in memory
2. **Fuse.js Integration**: Fuzzy matching for similar item detection
3. **User Confirmation**: Fuzzy matches require user verification
4. **Export Options**: Export matches, unique items, comparison summary
