# v2.0.0: Full Language Support

## Overview

Complete Itemdeck's internationalisation with RTL (Right-to-Left) language support, enabling Arabic and Hebrew users to enjoy a fully localised experience.

## Theme

**Global Accessibility** - Making Itemdeck usable by speakers of RTL languages with fully mirrored UI.

## Features

| ID | Feature | Priority | Complexity |
|----|---------|----------|------------|
| [F-077](../features/planned/F-077-rtl-support.md) | RTL Language Support | High | High |

## Scope

### Included

- Arabic translations (complete set)
- Hebrew translations (complete set)
- CSS logical properties migration
- RTL layout mirroring
- Icon mirroring for directional icons
- Language selector expanded

### Building On v1.5.0

This milestone requires v1.5.0's foundation:
- react-i18next infrastructure
- Translation namespace structure
- Icon registry with `mirrorInRTL` flags
- Language preference in settings store

## Technical Changes

### CSS Migration

All directional CSS properties converted to logical equivalents:

| Before | After |
|--------|-------|
| `margin-left` | `margin-inline-start` |
| `margin-right` | `margin-inline-end` |
| `padding-left` | `padding-inline-start` |
| `text-align: left` | `text-align: start` |
| `left` / `right` | `inset-inline-start` / `inset-inline-end` |

### New Locale Files

```
src/i18n/locales/
├── ar/                             # Arabic (RTL)
│   ├── common.json
│   ├── settings.json
│   ├── cards.json
│   ├── mechanics.json
│   ├── errors.json
│   └── accessibility.json
└── he/                             # Hebrew (RTL)
    └── ... (same structure)
```

### Modified Files

| File | Changes |
|------|---------|
| `src/App.tsx` | Set `dir` attribute on document root |
| `src/i18n/config.ts` | Add RTL locale detection |
| 100+ CSS module files | Convert to logical properties |
