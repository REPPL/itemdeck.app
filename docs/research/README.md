# Research Documentation

This directory contains research documents exploring design patterns, technical approaches, and best practices relevant to the ItemDeck project.

---

## Available Research

### [Card UI/UX Design Patterns](./card-ui-design-patterns.md)

Comprehensive research on modern card-based interface design, covering:

- **Card-based interfaces** - Core concepts, benefits, and when to use cards
- **Information density** - Balancing data richness with user comprehension
- **Progressive disclosure** - Revealing complexity on demand
- **Visual hierarchy** - Typography, spacing, colour, and depth techniques
- **Card stack and wallet metaphors** - Apple Wallet, physical metaphors, stacking patterns
- **Drag and drop patterns** - Visual feedback, handles, animations, accessibility
- **Responsive card sizing** - CSS Grid, Flexbox, mobile-first approaches
- **Visual design techniques** - Glassmorphism, gradients, shadows, elevation
- **Typography for cards** - Font selection, hierarchy, readability
- **Accessibility** - WCAG compliance, keyboard navigation, screen readers
- **Real application examples** - Apple Wallet, Google Pay, banking apps
