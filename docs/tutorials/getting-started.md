# Getting Started with itemdeck.app

Welcome to itemdeck.app! This tutorial will walk you through the basics of using the application.

## What is itemdeck.app?

itemdeck.app is a responsive card grid viewer for displaying collections. You can use it to browse games, books, media collections, or any other items you want to catalogue and explore.

## The Main Interface

When you first open itemdeck.app, you'll see:

1. **Card Grid** - The main area showing your collection as cards
2. **View Mode Toggle** - Bottom-left corner, switch between Grid/List/Compact views
3. **Floating Buttons** - Bottom-right corner, access to Search, Games, Settings, and Help

## Viewing Cards

### Flipping Cards

Click or tap any card to flip it. The front typically shows an image, while the back displays details about the item.

**Keyboard shortcut**: With a card selected, press `Space` to flip it.

### Card Detail View

Double-click (or double-tap) a card to open the full detail view. This shows all information about the item in an expanded overlay.

**Keyboard shortcut**: Press `Enter` to open detail view, `Escape` to close.

### Navigation

Use arrow keys to move between cards in the grid:
- `Arrow Left/Right` - Move horizontally
- `Arrow Up/Down` - Move between rows
- `Home` - First card
- `End` - Last card

## Changing the View

The view mode toggle (bottom-left when search is minimised) offers three layouts:

1. **Grid** - Cards in a responsive grid
2. **List** - Cards in a vertical list with more detail
3. **Compact** - Smaller cards for denser browsing

You can also change this in Settings (Quick tab > View Mode).

## Accessing Settings

Click the gear icon (bottom-right) or press `S` to open Settings. Here you can:

- Choose visual themes
- Adjust card sizes
- Enable/disable features
- Manage data sources

## Getting Help

- Press `?` (Shift + /) for keyboard shortcuts reference
- Click the Help button (question mark icon)
- Access this documentation
